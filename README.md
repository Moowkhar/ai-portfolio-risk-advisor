# AI Portfolio Risk Advisor

An AI-powered crypto portfolio risk analysis dashboard designed for Binance traders.

This tool analyzes a user's portfolio allocation and provides an instant risk assessment based on asset distribution, volatility exposure, and diversification.

## Live Demo
https://ai-portfolio-risk-advisor.vercel.app

## Features

* Portfolio allocation input (BTC, ETH, BNB, SOL, Other)

* AI-powered risk scoring

* Volatility and stability indicators

* Portfolio diversification analysis

* Binance-style trading dashboard interface

## How It Works

1. User inputs portfolio allocation percentages.
2. The AI risk engine evaluates diversification and exposure.
3. The system generates a portfolio risk score.
4. Users receive insights and recommendations to rebalance risk.

User Input → Risk Engine → Portfolio Analysis → Risk Score → Recommendations

## Use Case

Helps Binance traders quickly understand the risk level of their crypto portfolios before making trading decisions.
